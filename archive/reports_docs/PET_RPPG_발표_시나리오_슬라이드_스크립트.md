그래서 우리는 각 zone마다 Single과 Multi 중 어떤 것이 더 좋은지 **데이터로 판단**하는 Adaptive Selector를 만들었습니다."

## 발표 팁

- **강조할 키 메시지** (반복 추천):
  1. 픽셀 수가 정말 중요하다.
  2. 무조건 Multi가 아니라, **데이터로 판단**하는 Adaptive가 핵심.
  3. 강아지 전용 신호 모델 + Prior가 고심박에서 강력하다.

- **시간 배분 추천**:
  - 서론 + 전체 흐름: 5분
  - 단계별 상세 (특히 Adaptive + ROI 픽셀): 10~11분
  - 결과 + 결론: 5분

- **질문 대비 포인트**:
  - "왜 정사각형인가요?" → 현재는 단순성과 속도를 위해 square crop을 사용. 향후 circular mask + validity filter로 개선 예정.
  - "Ground truth가 정확한가요?" → 비디오 모니터 OCR 기반. ECG 수준은 아니지만, 상대 비교에는 충분히 의미 있음.
