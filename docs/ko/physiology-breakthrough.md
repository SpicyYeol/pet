# 생리학 기반 비접촉 심박 판별 — 기술 돌파 총정리

*English: [`docs/en/physiology-breakthrough.md`](../en/physiology-breakthrough.md)*

rPPG의 근본 난관(심박–아티팩트 선택 실패)을 **개의 생리(RSA)** 로 라벨 없이 절반 이하로 개선하고,
부수적으로 임상 신호(호흡 교차검증·미주신경·미세순환)를 확장했다. 모든 수치는 실측이며, 성공뿐 아니라
반례·한계까지 정직하게 기록한다.

## 1. 한 문단 요약

털·헐떡임·움직임 때문에 반려견 rPPG는 고심박에서 **~100 bpm 아티팩트에 고착**한다. 일반 신호품질
(SNR)로는 이 아티팩트가 오히려 "깨끗해 보여" 선택에 실패한다. 우리는 **정답 심박은 이미 영상에
존재**함을 규명(oracle MAE 0.4 bpm)하고, **개의 강한 호흡성 동성부정맥(RSA)** — 진짜 심박은 호흡
주파수로 변조되지만 아티팩트는 아님 — 을 판별 근거로 삼아 **held-out 심박 오차를 81.8 → 30.8 bpm
(−63%)** 로 개선했다. 라벨/학습 없이, **마취견 조건에서도** 달성됐고 기본 선택기로 승격됐다.

## 2. 문제와 진단 — "신호 부재가 아니라 선택 실패"

| 지표 | 결과 | 의미 |
|---|---|---|
| Oracle(정답 활용 상한) | **MAE 0.4 bpm**, 95% 일치 ±2 | 정답 심박이 후보에 **존재** |
| 정직한 held-out | MAE 45–82 bpm | **자동 선택이 미해결** |

병목은 신호가 아니라 **후보 선택**이다. 일반 라벨-프리 SQI도 MAE 49.5로 **전부 ~100bpm에 붕괴** —
아티팩트가 SNR·주기성·후보 합의 모두에서 진짜 심박보다 우세하기 때문. → 더 좋은 일반 품질로는 안 풀린다.
(근거: [`PRELIMINARY_VALIDATION`](../research/PRELIMINARY_VALIDATION.md))

## 3. 핵심 돌파 — RSA(호흡성 동성부정맥) 기반 선택

**생리**: 개는 RSA가 매우 강함 → 진짜 심박은 **호흡 주파수로 변조**, 동작/헐떡임 아티팩트는 안 됨.
**방법**: 후보별 **순간-HR 시계열**(스펙트로그램 → Viterbi 평활 → 서브빈 보간) 산출 → **RSA 결합**
(호흡대역 iHR 변조 파워 + 흉부 프록시 상관) 계산 → 생리적 타당범위 내 선택. 주파수(iHR) 기반이라
**RIIV(진폭 변조)가 아닌 진짜 RSA**를 겨냥.

| 선택 규칙 | 7영상 held-out MAE |
|---|---|
| SNR (일반 품질) | 81.8 bpm |
| RSA 결합 | 45.9 |
| **RSA + 생리 타당범위** | **30.8** ✅ |
| 기존 캐시 파이프라인 | 53.6 |
| oracle(상한) | 10.1 |

**SNR 대비 −63%, 기존 대비 −43%.** in-sample 최고치(37.5)보다도 낮음. **마취(RSA 둔화)인데도 작동** →
깨어있는 동물엔 더 나을 것. 기본 선택기로 승격, 영상 없으면 캐시 폴백.
(근거: [`RSA_SELECTOR_DESIGN`](../research/RSA_SELECTOR_DESIGN.md))

## 4. 상보적 판별 지표 (PLV·LF)

7종 생리 SQI 실험: RSA가 놓친 최난 케이스를 **고조파 위상잠금(PLV)·Mayer/LF 결합이 단독으로
복원**(오차 119→21) → 그 아티팩트가 *RSA 결합은 있으나 고조파·Mayer 구조가 없는* 호흡 진동임을 증명.
**물리적으로 상보적.** 단, n=7에서 손으로 융합하면 과적합 → 학습형 결합에는 라벨 데이터 필요.

## 5. 부수 성과 — 신규 임상 신호

| 신호 | 임상 의미 | 상태 |
|---|---|---|
| **RIIV → 호흡수** | 맥파에서 RR → 흉부 프록시와 **교차검증(신뢰도)** | 평균 Δ~10 brpm |
| **RSA 진폭** | HR 호흡변조 폭 = **미주신경 톤** | 방향성(노이즈로 과대) |
| **혈관운동(vasomotion)** | myogenic 대역 = **미세순환** | 신규(reference 필요) |
| **HRV Poincaré SD1/SD2·pNN50** | 자율신경 균형 | 반영 |

## 6. 정직한 반례·한계 (신뢰성의 근거)

- **A+B 추출 결합**: RSA에 더하니 **회귀(30.8→48.9)** — 고심박 맥파 약화. 미채택.
- **심탄도(BCG)**: 색이 아닌 움직임 독립 HR → 10/30fps 모두 **총체운동에 압도, 미작동**.
- **생리 특성 융합**: n=7 손튜닝은 과적합 → 학습형 선택기 + 데이터 필요.

## 7. 종합 스코어보드

| 시도 | 결과 |
|---|---|
| ✅ RSA 결합 | **돌파** (81.8→30.8, 기본값) |
| ✅ 알로메트리·발열·휴식·HRV·청색증-Hb | 반영 |
| ✅ RIIV·RSA진폭·혈관운동 | 신규 임상신호 |
| 🧪 PLV·LF (SQI 7종) | 상보적, 융합은 학습 필요 |
| ❌ A+B 결합·BCG·손튜닝 융합 | 미채택 |

## 8. 사업·기술적 의의

- **차별성**: rPPG를 신호품질이 아니라 **생리학(RSA·고조파·자율신경)으로 선택**. oracle 대비 격차 절반
  이하 실증. 검증의 정직성(oracle/held-out/overfit 구분, 반례 기록)으로 인허가·수의사 신뢰 확보.
- **데이터 논거**: 상보성·융합·미세신호 정밀검증이 모두 **동기화 reference(ECG/펄스옥시) 코호트**로 수렴 →
  무엇을·왜·얼마나 모아야 하는지 정량 제시.
- **확장성**: 산출된 9종 SQI + 신규 신호는 **학습형 다특성 선택기**의 입력이자 심부전·통증·미세순환
  모니터링의 임상 feature로 재활용.
