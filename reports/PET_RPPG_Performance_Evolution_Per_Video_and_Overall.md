# PET rPPG — 성능 진화 비교 (전체 데이터 + 개별 비디오)

**목적**: 발표에서 
- 전체 데이터셋 관점에서의 변화
- 개별 비디오(특히 중요한 비디오)에서의 변화

를 함께 보여주기 위한 자료.

## 1. 전체 데이터 관점 (7비디오 전체 Mean Error)

| Iteration | 주요 추가 | 7비디오 전체<br>Mean Error | 비고 (전체 데이터 관점) |
|-----------|-----------|-----------------------------|--------------------------|
| Baseline (Face-box) | Face-box + 기존 방법 | ~70~90+ | 대부분 불안정 |
| + Anatomical ROIs | DLC + 얇은 털 ROI | ~55~65 | 전체 후보 품질 향상 |
| + 생리적 신호 강화 (A+B) | 헐떡임 제거 + 심박 증폭 | ~45~55 | **가장 큰 단일 개선** |
| + Multi-Patch | Zone당 다중 패치 | ~40~48 | 전반적 노이즈 저항력 향상 |
| + Adaptive Selector | Per-zone 데이터 기반 선택 | ~38~42 | **전체 안정성 확보** |
| + **Dog-specific v1** | 강아지 전용 가중치 | **37.5** | **현재까지 최고 성능** |
| + Dog-specific v2 | 데이터 대량 확장 | 70.3 | 데이터 불균형으로 악화 (반례) |

## 2. 개별 비디오 관점 (주요 비디오별 변화)

### Video 7 (189.5 bpm) — 가장 어려운 고심박 케이스

| 단계 | Mean Error (이 비디오) | 비고 |
|------|-------------------------|------|
| Baseline | 90~100+ (100bpm lock) | 거의 항상 아티팩트에 고착 |
| + Anatomical + A+B | 40~60대 | 고심박 후보가 처음으로 안정적으로 검출 |
| + Multi-Patch + Adaptive | ~30~50 | 안정적 개선 |
| + **Dog-specific v1** | **21.3** | **이 비디오에서 최고 성능** |
| + Dog-specific v2 | 94.6 | 오히려 악화 (데이터 불균형 영향) |

**특이사항**: Prior-guided estimation 사용 시 error가 59.5 → 12.3까지 극적으로 떨어짐 (가장 강력한 후처리 효과 확인)

---

### Video 3 (210 bpm) — 가장 높은 심박수

| 단계 | Mean Error (이 비디오) | 비고 |
|------|-------------------------|------|
| Baseline | 100+ (100bpm lock) | 완전 실패 |
| + Anatomical + A+B | 40~60대 | 170~210+ bpm 후보 등장 |
| + Multi-Patch + Adaptive | ~40~55 | 추가 개선 |
| + **Dog-specific v1** | ~49.5 | 안정적 회복 |
| + Dog-specific v2 | 122.7 | 크게 악화 |

---

### Video 6 (90 bpm) — 저심박 비디오 (Adaptive의 중요성을 보여준 케이스)

| 단계 | Mean Error (이 비디오) | 비고 |
|------|-------------------------|------|
| Baseline | 중간 수준 | - |
| + Multi-Patch (강제 적용 시) | 크게 악화 | Multi-Patch를 무조건 쓰면 저심박에서 오히려 나빠짐 |
| + **Adaptive Selector** | 크게 개선 | Per-zone 선택으로 저심박 성능 회복 |
| + Dog-specific v1 | 양호 | 전체적으로 안정 |

**특이사항**: Adaptive Selector가 없었다면 전체 데이터 평균이 더 나빠졌을 가능성이 높음.

---

### 기타 비디오 (1,4,5,8) 요약

- 대부분의 저~중심박 비디오에서는 **Anatomical ROI + A+B** 단계에서 이미 상당한 개선이 있었음.
- **Multi-Patch**와 **Adaptive Selector**는 "고심박 비디오를 해치지 않으면서 저심박 비디오도 안정적으로 유지"하는 데 결정적 역할을 함.
- Dog-specific v1은 전체적으로 안정적이거나 소폭 개선되는 경향.
